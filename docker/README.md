# Запуск приложения в Docker

---

## Подготовка

Для начала необходимо скопировать в директорию
с этим README.md нейронку (swa\_convnext.pt) и секреты (env.sh).

Их можно найти у кого-то из нас или в группе в тг.

Затем убедитесь, что Docker Desktop работает.

Откройте этот каталог через GIT bash console или ещё как-нибудь, чтобы был доступ к линуксовым командам.
Убедитесь, что env.sh и startup.sh исполняемые или, если не уверены, выполните:
```bash
chmod +x env.sh && chmod +x startup.sh
```

## Запуск

Чтобы запустить загрузку репозиториев, сборку и запуск в Docker нужно выполнить простую
команду:
```bash
source startup.sh <path_to_java_sdk>
```
